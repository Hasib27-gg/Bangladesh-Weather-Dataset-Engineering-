# Overall requirements

## \[1\] Base Setup:
- Any code editor (like VS code, sublime text etc).
- Python & library setup (scroll down bellow to get description).
- An empty folder at the starting.
- All the databases , datasets and the main.ipynb in the same folder.

## \[2\] Python Setup:
- Must have Python 3.12+ 
- Must have libararies: Sklearn, XGBoost, LightGM, CatBoost, Optuna.
- If you don't have those libararies, run `pip install -r requirements.txt` in your terminal.
